# Design System

> **Status:** Phase 0 — design tokens defined in `theme.json` and CSS custom properties.

## Color Palette

| Token                     | Value     | Usage           |
| ------------------------- | --------- | --------------- |
| `--nb-color-text`         | `#1a1a1a` | Body text       |
| `--nb-color-text-muted`   | `#5c5c5c` | Meta, captions  |
| `--nb-color-background`   | `#ffffff` | Page background |
| `--nb-color-surface`      | `#f5f5f5` | Footer, cards   |
| `--nb-color-accent`       | `#2563eb` | Links, buttons  |
| `--nb-color-accent-hover` | `#1d4ed8` | Hover states    |

## Typography

- **Font stack:** system-ui, Segoe UI, Roboto, sans-serif
- **Base size:** `1rem` / line-height `1.6`
- **Container max-width:** `72rem`

## Spacing Scale

| Token             | Value    |
| ----------------- | -------- |
| `--nb-spacing-xs` | `0.5rem` |
| `--nb-spacing-sm` | `1rem`   |
| `--nb-spacing-md` | `1.5rem` |
| `--nb-spacing-lg` | `2.5rem` |
| `--nb-spacing-xl` | `4rem`   |

## Block Editor

Design tokens are mirrored in `theme.json` for Gutenberg parity.

## Future

Tailwind CSS (`tailwind.config.js`) is pre-configured for the React builder UI in Phase 2.
