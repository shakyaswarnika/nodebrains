A lightweight, modular, high-performance custom WordPress theme built for a future proprietary visual page builder.

== Description ==

NodeBrains is a classic PHP WordPress theme with a modular architecture, block editor support via theme.json, and scaffolding for a custom visual page builder.

= Features =

* Responsive layout with semantic HTML5 markup
* Custom logo, header image, and background support
* Primary and footer navigation menus
* Main sidebar and footer widget areas
* Block editor styles and wide alignment support
* Customizer copyright text with live preview
* Full-width page template
* Translation ready

== Installation ==

1. Upload the `nodebrains` folder to `/wp-content/themes/`.
2. Activate the theme through the **Appearance → Themes** screen.
3. Configure menus under **Appearance → Menus**.
4. Add widgets under **Appearance → Widgets**.

== Frequently Asked Questions ==

= Does this theme require a page builder? =

No. NodeBrains does not depend on Elementor or any third-party page builder.

= Where are theme styles loaded? =

Front-end styles are enqueued from `assets/css/main.css`.
