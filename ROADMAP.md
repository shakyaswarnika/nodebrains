# NodeBrains Roadmap

High-level product roadmap for the NodeBrains WordPress theme and proprietary visual page builder.

## Phase 0 — Project Foundation ✅ (Current)

- [x] WordPress theme skeleton
- [x] Git + GitHub repository
- [x] Composer (PHPCS, WPCS, PHPCompatibility)
- [x] npm toolchain (ESLint, Prettier, Husky, Commitlint)
- [x] GitHub Actions CI/CD
- [x] Documentation structure

## Phase 1 — Theme Hardening

- [ ] PHPUnit test suite
- [ ] Theme Check automation in CI
- [ ] Accessibility audit (WCAG 2.1 AA)
- [ ] Performance baseline (Lighthouse CI)
- [ ] Expanded block patterns
- [ ] RTL stylesheet

## Phase 2 — Builder Architecture

- [ ] Next.js app scaffold in `apps/builder/`
- [ ] Module registry implementation
- [ ] REST API endpoints for builder data
- [ ] Server-side render pipeline
- [ ] Builder admin UI shell (React + TypeScript)

## Phase 3 — Visual Page Builder MVP

- [ ] Drag-and-drop canvas
- [ ] Core modules: Section, Column, Heading, Text, Image, Button
- [ ] Live preview integration
- [ ] Save/load page layouts to post meta
- [ ] Responsive breakpoint controls

## Phase 4 — Builder Expansion

- [ ] Template library
- [ ] Global styles / design tokens sync
- [ ] Dynamic content (post loops, ACF)
- [ ] Import/export layouts
- [ ] Role-based editor permissions

## Phase 5 — Productization

- [ ] Plugin distribution option
- [ ] Marketplace-ready packaging
- [ ] Automated release pipeline
- [ ] Documentation site
- [ ] E2E tests (Playwright)

---

**Vision:** A lightweight WordPress theme with a proprietary, React-powered visual builder that rivals commercial solutions without third-party page builder dependencies.

Timeline is flexible and driven by milestone completion, not fixed dates.
